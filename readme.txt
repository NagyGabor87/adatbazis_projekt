A megfelelő futtatáshoz hozd létre 
- telepitsd a requirements.txt fájlban lévő függőségeket
- a project_folder.info szerinti mapparendszert a py projekten belül
- másold be a nyers .csv fájlokat az import mappába
- inditsd el a main.py fájlt

Az adatbázis automatikusan létrejön a projekt db mappájában, illetve ha a nyers .csv fájlok dekódolása sikerült, akkor azok a temp mappába kerülnek, majd tisztitás után az export mappába.
A program a tisztitott adatokat az expot mappából tölti be az adatbázisba.
